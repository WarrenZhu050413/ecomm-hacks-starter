Chromatic Aberration

RGB color channels split apart, creating that classic camera lens distortion effect. Popular in cyberpunk aesthetics.

/* Stack 3 copies, offset red/blue channels */ .red { transform: translateX(-3px); mix-blend-mode: screen; } .blue { transform: translateX(3px); mix-blend-mode: screen; }
"Symmetry reveals hidden patterns."
Technique 2 of 12

Kaleidoscope

Mirror the image in quadrants to create mandala-like symmetry. Hypnotic and meditative.

/* 4 copies with different transforms */ img:nth-child(2) { transform: scaleX(-1); } img:nth-child(3) { transform: scaleY(-1); } img:nth-child(4) { transform: scale(-1); }
"Electric dreams in digital nights."

Neon Glow / Cyberpunk

High contrast with saturated neon color overlay using color-dodge blend mode. Pure synthwave energy.

img { filter: brightness(0.3) contrast(2) saturate(3); } ::after { background: linear-gradient(135deg, #f0f, #0ff, #f08); mix-blend-mode: color-dodge; }
"Through the looking glass."

Inverted / Negative

Classic photo negative effect. Invert colors then hue-rotate to maintain some tonal relationships.

filter: invert(1) brightness(0.8) hue-rotate(180deg);
"Where light becomes shadow."

Glitch Effect

Horizontal slice displacement with hue shifts. Simulates corrupted video data. Add animation for extra chaos.

/* Stack images with clip-path slices */ .layer:nth-child(2) { clip-path: polygon(0 20%, 100% 20%, 100% 25%, 0 25%); transform: translateX(10px); filter: hue-rotate(90deg); }
"Heat signatures of the soul."

Thermal / Heat Map

Maps luminosity to a heat color gradient. Blue=cold, red=hot. Scientific meets psychedelic.

img { filter: grayscale(100%) contrast(1.5); } ::after { background: gradient(blue ‚Üí cyan ‚Üí green ‚Üí yellow ‚Üí red); mix-blend-mode: color; }
"Light through a crystal mind."

Prismatic / Rainbow

Animated rainbow gradient overlay. The hue-rotate animation creates constantly shifting colors.

::after { background: linear-gradient(ROYGBIV colors); mix-blend-mode: overlay; animation: hue-rotate 10s infinite; }
"Reduced to essence."

Pixelate

Scale down then up with nearest-neighbor rendering. Creates that classic 8-bit retro game aesthetic.

img { image-rendering: pixelated; width: 25%; height: 25%; transform: scale(6); }
"Strokes of digital impasto."

Oil Painting

High saturation + slight blur + turbulence texture overlay. Gives painterly, Van Gogh-like quality.

img { filter: saturate(2.5) contrast(1.3) blur(1px); } ::after { background: SVG turbulence noise; mix-blend-mode: soft-light; }
"Iridescent dimensions."
